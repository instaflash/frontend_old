### Configure GIT
```
cp ./.gitconfig $HOME/
```
---
### Install NPM packages
```
yarn install
```
---
### Start development
```
PORT=8080 yarn start
```
---
### TypeScript lint
```
yarn run tslint -p ./
```
---
### Build
```
yarn run build
```
---
### Run HTTP server
```
PORT=80 yarn run server
```
